Feature
- UI เต็มรูปแบบ
- แอนิเมชั่นของไพ่
- โมเดลโต๊ะลายพิเศษที่ทำมาเพื่อบาคาร่า
- โชว์ผลลัพธ์ผ่าน 3D Text ของโต๊ะให้ผู้ชม
- สามารถคุยระหว่างอยู่ในโต๊ะได้
- ไม่จำกัดผู้เล่นต่อโต๊ะ
- โต๊ะที่สามารถปรับแต่งได้มากกว่าเดิม
- Pattern สำหรับปรับเรทการชนะได้ในตัว
- Reward Multipy สำหรับปรับเรทรางวัล
- Discord Webhook ที่จะเก็บทุก Log ของผู้เล่นที่มาเล่นไว้
- การทำงานของ Logic ที่อิงมาจากคาสิโนออนไลน์ของจริง
- ระบบการทำงานที่เร็วกว่าเดิม

รายการที่ปรับแต่งได้
- เพิ่มโต๊ะได้เรื่อยๆ
- ปรับแต่งโต๊ะได้
- ปรับอัตราคูณเงินรางวัลได้
- ปรับขั้นต่ำและสูงสุดของแต่ละโต๊ะได้
- ปรับแต่งเวลาในโต๊ะ
- ปรับภาษาได้

รูปแบบการเล่นจะอิงตามกฏของบาคาร่า รวมทั้งการบังคับจั่วไพ่เพิ่ม

**วิธีติดตั้ง**
1. โยนลง resources ของ fivem
2. เพิ่ม start scotty-casino-baccarat และ start scotty-casino-baccarat-props ลงใน server.cfg
3. ใส่ license key ที่ได้จากผู้ขายใส่ลงในไฟล์ config_sv.lua
4. ปรับแต่งของที่สามารถทำได้ในไฟล์ config.lua
